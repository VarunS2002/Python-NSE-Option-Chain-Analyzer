> # [4.0](https://github.com/VarunS2002/Python-NSE-Option-Chain-Analyzer/releases/tag/4.0)

## Major Feature Update

- Added support for FINNIFTY index

- Improved Buttons
- Modified some Labels

        * Logging Started
        * Logging Stopped
        * Program Quitting
    - Removed unnecessary messages:
